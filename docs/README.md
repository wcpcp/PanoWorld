## GitHub Pages

For the `wcpcp/PanoWorld` repository, the simplest setup is:

1. Copy this whole `docs/` directory into the repo root
2. Push to the default branch
3. In GitHub repository settings, enable Pages from the `/docs` folder

Then the page should be served automatically.
